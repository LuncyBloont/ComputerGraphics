# 新的OpenGL (programmable architecture using shaders)
## 摘要
最新的OpenGL Core已经移除了固定渲染管线的内容，glBegin(...)...glEnd();的用法已经过时，最新的OpenGL程序一般使用shader、VAO和VBO等对象让GPU使用顶点数据和着色器绘图。<br>
